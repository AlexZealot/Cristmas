# Cristmas

Cristmas illumination
Used librariies:
1) FastLED
2) AsyncUDP
3) TFT_eSPI

Components for lighting obects:
1) Webos D1 mini or LOLIN S2 mini
2) resistor
3) LED strip or garland based on WS2812b
4) push-buttons (1-2)
5) optionally BME280
6) TFT touch screen (I used 2.8' based on ILI9341)
